# Todo Web App (Backend)

A simple backend API for managing personal to-do lists.  
Each user has their own tasks and cannot access others' todos.

Built with Django + Django REST Framework. JWT is used for authentication, and Swagger provides API documentation.

## Features
- User registration and authentication
- CRUD operations for todos
- JWT-based authentication
- Swagger UI for API testing

## Tech Stack
- Python 3.13
- Django 6.0
- Django REST Framework
- drf-yasg (Swagger)
- djangorestframework-simplejwt (JWT)
